# Папка для изображений

Поместите вашу фотографию для первого экрана в эту папку с именем `hero-photo.jpg`.

Рекомендуемые параметры изображения:
- Формат: JPG или PNG
- Разрешение: минимум 1920x1080 пикселей (для лучшего качества)
- Ориентация: горизонтальная (landscape) или квадратная
- Размер файла: рекомендуется до 2MB для быстрой загрузки

Если ваше изображение имеет другое имя или формат, обновите путь в файле `index.html` в строке:
```html
<img src="images/hero-photo.jpg" alt="Иван и Елизавета" class="hero-image">
```
